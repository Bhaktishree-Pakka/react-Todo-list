# React Todo-list

A modern, fully functional React Todo List app with dark mode, task filtering, edit/delete functionality, and persistent UI.

---

## 🚀 Live Demo

https://react-todo-list-psi-umber.vercel.app/

---

## ✨ Features

* ✅ Add new tasks  
* ✏️ Edit tasks  
* ❌ Delete tasks  
* 🔄 Mark tasks as complete  
* 🎯 Clean and minimal UI  
* ⚡ Fast performance using Vite

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Bhaktishree-Pakka/react-todo-list.git
```

2. Navigate to the project folder:

```
cd react-todo-list
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
